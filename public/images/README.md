# Images Directory

This directory should contain all the images for the Shubbies Catering and Decor website.

## Required Images

### Hero Slideshow (Landing Page)
- `slide1.jpg` - Main hero image showcasing elegant catering
- `slide2.jpg` - Decor setup image
- `slide3.jpg` - Premium service image

### About Section
- `about.jpg` - Image representing the business/team

### Portfolio Section
- `portfolio1.jpg` - Wedding reception example
- `portfolio2.jpg` - Corporate event example
- `portfolio3.jpg` - Birthday celebration example
- `portfolio4.jpg` - Garden party setup
- `portfolio5.jpg` - Traditional ceremony
- `portfolio6.jpg` - Luxury banquet

## Image Specifications

- **Format**: JPG or PNG
- **Recommended Size**: 
  - Hero slides: 1920x1080px (landscape)
  - About image: 800x600px
  - Portfolio images: 800x600px (landscape)
- **Quality**: High quality, optimized for web (compressed but clear)
- **Style**: Professional, elegant, showcasing the quality of your work

## How to Add Images

1. Place your images in this directory (`public/images/`)
2. Name them exactly as specified above
3. The website will automatically display them
